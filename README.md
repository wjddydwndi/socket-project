# socket-project


프로젝트 주제	: Java Socket통신을 사용한 응용프로그램 만들기.<br>
<br>
프로젝트 설명 : 로그인 처리를 통하여, 유저마다 자신의 사진 및 글이 담긴 게시물을 게시.<br>
각 게시물마다 로그인 계정으로 댓글 작성 가능.<br>
<br>

개발 툴	: Eclipse<br>
<br>
개발 기술 : JAVA, Oracle, mybatis<br>
<br>
프로젝트 기간	: 2019. 05 .15 ~ 2019.05.25 <br>
